The output of this section is configured in the tagscloud.tpl file. This section configures the templates used to work with the tag cloud. This block displays all keyword tags added to news items.

The following tags can be used:

{tags} - Used to output all keyword tags stored in the database

{tags limit="100"} - Used to output the 100 most popular keyword tags stored in the database
