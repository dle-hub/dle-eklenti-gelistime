In this section, you can configure the page that will be displayed when the website is disabled, for example for maintenance. Remember that administrators will always be allowed to view the website as if it were enabled. Everyone else will see the page from this section.

The template is configured in the offline.tpl file and the following tags can be used:

{reason} - Used to output the reason for disabling the website that you specify in the script settings
