The output of this section is configured in the vote.tpl file. This section configures the templates used for website voting.

The following tags can be used:

{title} - Poll title

{list} - List of questions

{vote_id} - Poll ID

{votes} - Number of votes

[votelist] text [/votelist] - Outputs the text that should be shown to people who have not voted yet

[voteresult] text [/voteresult] - Outputs the text that should be shown to people who have already voted
