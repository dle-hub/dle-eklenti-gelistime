The output of this section is configured in the preview.tpl file. This section configures templates intended for previewing added news items. In this template, you can use all tags for short news, full news, and static pages. To separate the displayed parts, the following tags are additionally used:

[short-preview] text [/short-preview] outputs the template text intended for viewing the short news item

[full-preview] text [/full-preview] outputs the template text intended for viewing the full news item

[static-preview] text [/static-preview] outputs the template text intended for viewing the static page
