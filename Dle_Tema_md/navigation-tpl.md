The output of this section is configured in the navigation.tpl file. This section configures navigation for news.

[prev-link]Link text[/prev-link]

Creates a link to the previous page (if one exists)

[next-link]Link text[/next-link]

Creates a link to the next page (if one exists)

#### TAG: `{pages}`

Creates links to pages; for example: 1 2 3 4
